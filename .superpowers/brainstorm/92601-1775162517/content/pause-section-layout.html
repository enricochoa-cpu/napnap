Design Notes

?

Duration label shows "net" hint

When pauses exist, the duration label shows the net value with a subtle "(net)" suffix so parents know pauses were subtracted. When no pauses, no suffix — just the plain duration.

?

Collapsed card = one-line summary

Each pause card shows "Pause N" + start time + duration on one line. Tap to expand and edit the start time and duration inputs. Delete icon always visible.

?

Duration picker (not end time)

Matches Napper's pattern and the spec — parents think "baby woke for 10 minutes" not "woke at 15:20, slept again at 15:30." Simpler mental model at 3AM.
